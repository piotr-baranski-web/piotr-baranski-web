## Jak wgrać na GitHub

### Opcja 1: Przez GitHub Web Interface

1. Przejdź na https://github.com i zaloguj się
2. Utwórz nowe repozytorium o nazwie `piotr-baranski-web`
3. Kliknij "uploading an existing file"
4. Przeciągnij wszystkie pliki z folderu `portfolio na git` do przeglądarki
5. Kliknij "Commit changes"

### Opcja 2: GitHub Desktop (jeśli masz)

1. Pobierz GitHub Desktop: https://desktop.github.com/
2. Zaloguj się i dodaj repozytorium
3. Skopiuj pliki do folderu repozytorium
4. Zrób commit i push

## Włączenie GitHub Pages

1. W repozytorium na GitHub przejdź do **Settings**
2. Przejdź do sekcji **Pages** (po lewej)
3. W **Source** wybierz **Deploy from a branch**
4. Wybierz branch **main** i folder **/ (root)**
5. Kliknij **Save**
6. Po chwili Twoja strona będzie dostępna pod adresem: `https://twoj-login.github.io/piotr-baranski-web/`

## Konfiguracja formularza kontaktowego

Formularz używa Formspree do wysyłania wiadomości:

1. Przejdź na https://formspree.io i załóż darmowe konto
2. Utwórz nowy formularz
3. Skopiuj URL formularza (np. `https://formspree.io/f/abc123xyz`)
4. W pliku `index.html` znajdź linię z `action="https://formspree.io/f/your-form-id"`
5. Zamień `your-form-id` na ID z Twojego formularza Formspree
6. W ustawieniach formularza w Formspree ustaw email: `piotrbaranski96@gmail.com`
